Zum Spielstart main.py ausführen.

M 		      -> Musik stummschalten
SPACE 		      -> Ball wegschießen mit Plunger
A / linke Pfeiltaste  -> Linker Flipper
D / rechte Pfeiltaste -> Rechter Flipper
F2 		      -> Spiel zurücksetzen
ESC		      -> Spiel verlassen

Wenn alle drei Tränke eingesammelt -> x2 Punkte für 10 Sekunden
